# 數學專題：費波那契數列與黃金比例的奧秘

## 1. 專案簡介
本專案為期中作業，旨在透過 Python 程式實作來探索經典數學問題——**費波那契數列 (Fibonacci Sequence)** 與 **黃金比例 (Golden Ratio, $\phi$)** 之間的動態關係。

## 2. 數學原理
費波那契數列的定義如下：
- $F_0 = 0$
- $F_1 = 1$
- $F_n = F_{n-1} + F_{n-2}$ (當 $n \ge 2$)



[Image of Fibonacci spiral and golden ratio]


本專案的核心在於驗證以下極限：
當 $n$ 趨向無窮大時，相鄰兩項的比值會收斂於黃金比例：
$$\lim_{n \to \infty} \frac{F_n}{F_{n-1}} = \phi = \frac{1 + \sqrt{5}}{2} \approx 1.6180339887...$$

## 3. 程式功能說明
`fibonacci_math.py` 具有以下功能：
1. **序列生成**：根據使用者輸入的項數，動態計算費波那契數列。
2. **比例計算**：自動計算每一項與前一項的比值。
3. **誤差分析**：將最後計算出的比值與理論上的黃金比例數值進行對比，計算其絕對誤差，展示收斂的精確度。
