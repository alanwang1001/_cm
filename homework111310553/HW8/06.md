# 資訊理論核心：夏農兩大定理說明文件

這份文件詳細說明了通訊領域的兩大基石：**夏農信道編碼定理**與**夏農-哈特利定理**。

---

## 1. 夏農信道編碼定理 (Shannon's Channel Coding Theorem)
又稱為**有噪通道編碼定理 (Noisy-Channel Coding Theorem)**。

### 核心定義
該定理指出：對於任何給定的通道，都存在一個確定的數值，稱為**通道容量（Channel Capacity, $C$）**。
* 如果傳輸速率 $R < C$，則存在一種編碼方式，使得傳輸錯誤率趨近於 **0**。
* 如果 $R > C$，則無論如何設計編碼，錯誤率都無法降至零。

### 直觀理解
想像通道是一條「高速公路」，通道容量 $C$ 是這條路每小時能承載的最大車流量。
* **只要車流量低於限度**：透過良好的交通調度（編碼），車輛可以完全不相撞。
* **一旦車流量超載**：車禍（錯誤）就不可避免。

> **歷史意義**：它打破了「傳輸越快，錯誤一定越多」的直覺，證明了只要在容量範圍內，我們可以達到**既快又無誤**的通訊。

---

## 2. 夏農-哈特利定理 (Shannon–Hartley Theorem)
這是信道編碼定理在「連續訊號（如無線電、電話線）」中的具體數學應用，用來計算極限頻寬。

### 數學公式
在具有加性高斯白雜訊 (AWGN) 的通道中，通道容量 $C$ 的計算公式如下：

$$C = B \log_2 \left( 1 + \frac{S}{N} \right)$$

**參數說明：**
* **$C$**: 通道容量 (bits per second, bps)
* **$B$**: 頻寬 (Bandwidth, Hz)
* **$S$**: 訊號功率 (Signal Power)
* **$N$**: 雜訊功率 (Noise Power)
* **$S/N$**: 信噪比 (Signal-to-Noise Ratio, SNR)



### 關鍵因素分析
1.  **頻寬 ($B$)**: 容量與頻寬成正比。增加頻寬是提升速度最直接的方法（如 5G 使用毫米波）。
2.  **信噪比 ($S/N$)**: 雜訊越強，可用的有效資訊位元就越少。

---

## 3. 定理對比表

| 特性 | 夏農信道編碼定理 | 夏農-哈特利定理 |
| :--- | :--- | :--- |
| **關注點** | 傳輸的**可靠性**（錯誤率） | 傳輸的**極限速度**（容量） |
| **主要結論** | 速率 $R < C$ 時，錯誤可為零 | 給出特定物理環境下 $C$ 的具體數值 |
| **應用範疇** | 糾錯碼（Hamming, LDPC, Polar Code） | 行動通訊、光纖、Wi-Fi 規格設計 |

---

## 4. 總結
* **信道編碼定理**告訴我們：**「完美的傳輸是可能的」**。
* **夏農-哈特利定理**告訴我們：**「物理極限在哪裡」**。
